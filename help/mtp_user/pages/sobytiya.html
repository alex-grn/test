События

Раздел события – список задач к коллективному исполнению. В разделе предусмотрены следующие статусы:

"Новое" – статус присваивается автоматически при добавлении записи.

"В работе" – статус присваивается по действию при формулировке  события.

"На уточнении" – статус присваивается по действию при уточнении задачи события.

"На доработке" – статус присваивается по действию при изменении задачи  события.

"Исполнено" – статус присваивается по действию при реализации события. Запись становится только на чтение.

"Аннулировано"   – статус присваивается по действию при отмене события. Запись становится только на чтение.

2.2 Вход в систему

Для начала работы с модулем в строке любого браузера ввести адрес, полученный у администратора Системы. Откроется окно авторизации (Рисунок 1). Ввести логин пользователя и пароль, нажать «ОК».

 

Рисунок 1 - Окно авторизации

 

Если Логин (пароль) не соответствует зарегистрированным данным, Система выдаст сообщение (Рисунок 2).

 

Рисунок 2 - Вход в систему не возможен

 

При правильном наборе данных, откроется главное окно модуля (Рисунок 3).

 

Рисунок 3 - Главное окно модуля

 

Важно! Для разных ролей видимость разделов отличается.

Администратором АИС РПП для каждого пользователя назначается временный пароль со сроком давности. По завершении срока действия пароля, доступ в Систему будет заблокирован (Рисунок 4). Доступ возможен только для смены пароля.

 

Рисунок 4 - Окончание срока действия временного пароля

 

Важно! При первом запуске системы обязательно сменить пароль пользователя.

Для смены пароля последовательно прейти «Система» а «Изменить пароль» (Рисунок 5).

 

Рисунок 5 - Меню раздела «Система»

 

В открывшейся форме (Рисунок 6) заполнить все поля и нажать «ОК». Для последующего входа (Рисунок 7) использовать новый пароль. В случае утери пароля – обратиться к администратору АИС РПП для повторного назначения временного пароля.

 

Рисунок 6 - Смена пароля пользователя

 

Рисунок 7 - Подтверждение о смене пароля

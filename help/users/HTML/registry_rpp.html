2.4.2 Реестры

Данный подраздел информационный.

Пользователь с ролью «Субъект» в данном разделе может:

-просматривать всю информацию по загруженным реестрам (только свой регион):

a)сведения, содержащиеся в загруженных реестрах;

b)сведения по получателям пособий;

c)сведения по детям получателей пособий;

d)сведения о выплатах, доплатах и удержаниях.

Пользователь с ролью «Сотрудник» и выше в данном разделе имеет все полномочия, указанные выше, за исключением того, что для просмотра доступны все регионы.

Все действия, описанные ниже для данного подраздела идентичны. Отличаться могут только содержащиеся в них сведения.

Для работы с подразделом выбрать в главном меню пункт «Реестры» а «Реестры» (Рисунок 18). В зависимости от интересующей информации, выбрать необходимый реестр.
